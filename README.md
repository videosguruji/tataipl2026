# CricPulse.ai | Personalized Tata IPL 2026 AI Companion

CricPulse.ai is a state-of-the-art, personalized AI content discovery platform built for the Tata IPL 2026 season. Designed specifically to eliminate content overload for diverse types of fans, this prototype demonstrates how Google Gemini's cognitive models can tailor live dashboards, match narratives, smart conversational answers, and interactive simulation widgets based on the user's specific persona.

## 🌟 Persona Options in Action
- **Gill's Devotee (GT Fan)**: Highly energetic, GT-centric news, Gill statistics, and playoff calculators.
- **The Tactician (Fantasy Player)**: Highly analytical, pitch indices, player-vs-player ratios, and emerald green dark mode visual branding.
- **The Express Fan (Casual Viewer)**: Excitement-first, short summaries (under 50 words), and bold highlighted key moments.
- **The Data Geek (Stats Lover)**: Dense metrics, strike rate bar graphs, bowler economy matrices, and structured data tables.

## 🛠️ File Structure
- `index.html`: Stunning glassmorphism UI layout featuring responsive columns, status indicators, and SVG assets.
- `styles.css`: High-fidelity dark mode stylesheet utilizing CSS variables, glowing borders, custom responsive grids, and CSS animation frames.
- `app.js`: Reactive javascript controller running local context datasets, AI response simulators, and interactive widgets.

## 🚀 How to Run Locally
1. Start a local static file server using any preferred tool, for example, using Python:
   ```bash
   python3 -m http.server 8000
   ```
2. Open your browser and navigate to `http://localhost:8000`.
3. Toggle between the persona buttons at the top of the header to observe the instant design system shifts, widget switches, and prompt injection updates!
